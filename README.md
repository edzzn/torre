# Torre

This projects implements [Torre.co](https://torre.co/) Bio API

## Live preview

[https://torre.now.sh/](https://torre.now.sh/)

## About

This is a React app.

It uses:

- Redux-Sagas
- Styled components

## Usage

Clone the project

```
git clone https://github.com/edzz/torre.git
cd ./torre
```

Install required dependencies:

```
npm install
```

Start the project:

```
npm start
```
